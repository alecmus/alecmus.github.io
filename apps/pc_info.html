PC Info
View essential PC specs in a snap

A simple tool for viewing essential PC specs. The app displays the most essential specifications of a PC in a simple and straightforward layout.

Features
View battery health, info about the CPU, graphics (GPUs, monitors), RAM and storage on a single clean form. Install or run in portable mode, copy or export specs to file and receive the latest updates from within the app.

System Requirements
Windows 10, Windows 11 or later
1280x720 screen resolution (HD) or more
20MB disk space or more
64MB RAM or more
